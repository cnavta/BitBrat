### Features

**OAuth Scopes (21 total):**
- Core messaging: `app_mentions:read`, `channels:history`, `chat:write`, `chat:write.public`
- Direct messages: `im:history`, `im:read`, `im:write`
- Private channels: `groups:history`, `groups:read`, `groups:write`
- Group DMs: `mpim:history`, `mpim:read`, `mpim:write`
- User context: `users:read`, `users:read.email`
- Reactions: `reactions:read`, `reactions:write`
- Files: `files:read`, `files:write`
- Customization: `chat:write.customize`
- Team info: `team:read`

**Event Subscriptions (13 events):**
- `app_mention` - @BitBrat mentions (primary trigger)
- `message.channels` - Public channel messages
- `message.groups` - Private channel messages
- `message.im` - Direct messages
- `message.mpim` - Group DMs
- `reaction_added`, `reaction_removed` - Sentiment tracking
- `user_change` - User profile updates
- `team_join` - New workspace members
- `channel_created`, `channel_deleted`, `channel_rename` - Channel management
- `member_joined_channel`, `member_left_channel` - Channel membership

**Architecture:**
- Dual-mode connector (Socket Mode + Events API)
- Socket Mode enabled (primary ingress, WebSocket)
- Events API webhook configured (fallback/redundancy)
- Webhook URL: `https://bitbrat.ai/webhooks/slack`
- Bot always online
- App Home enabled (messages tab + home tab)

**Security:**
- Webhook signature verification (HMAC-SHA256)
- Replay attack prevention (5-minute timestamp window)
- Timing-safe signature comparison
- Bot message filtering (loop prevention)

### Usage

**Create Slack App:**
1. Go to https://api.slack.com/apps
2. Click "Create New App" → "From an app manifest"
3. Choose workspace
4. Paste contents of `slack-app-manifest.yaml`
5. Review and create

**Configure Tokens:**
```bash
# Local (.env)
SLACK_APP_TOKEN=xapp-1-...
SLACK_BOT_TOKEN=xoxb-...
SLACK_SIGNING_SECRET=...
SLACK_ENABLED=true

# Production (Secret Manager)
gcloud secrets create SLACK_APP_TOKEN --data-file=-
gcloud secrets create SLACK_BOT_TOKEN --data-file=-
gcloud secrets create SLACK_SIGNING_SECRET --data-file=-
```

**Deploy:**
```bash
npm run brat -- deploy service ingress-egress
```

**Test:**
```slack
/invite @BitBrat
@BitBrat hello
```
